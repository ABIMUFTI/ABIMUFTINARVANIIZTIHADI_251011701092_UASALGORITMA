 Program Gaji Karyawan (C++)

Program ini merupakan program C++ sederhana untuk menghitung dan menampilkan
data gaji karyawan berdasarkan jam kerja dan jam lembur.

Program menggunakan konsep dasar bahasa C++ seperti variabel, konstanta,
array, perulangan, dan output berbentuk tabel.

## Deskripsi Program
Program menyimpan data beberapa karyawan dalam array, seperti:
- Nama karyawan
- Jumlah jam kerja

Program kemudian menghitung:
- Honor kerja berdasarkan jam kerja
- Honor lembur jika jam kerja melebihi batas tertentu
- Total honor yang diterima karyawan

Hasil perhitungan ditampilkan dalam bentuk tabel agar mudah dibaca.

## Fitur Program
- Menampilkan data karyawan
- Menghitung honor harian
- Menghitung honor lembur
- Menampilkan hasil dalam format tabel

- C++
